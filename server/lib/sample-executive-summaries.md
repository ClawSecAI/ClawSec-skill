## Usage in Reports

The executive summary appears at the top of each security report, before the detailed technical findings. This allows executives to:

1. **Quickly understand** the overall security posture
2. **Make decisions** about resource allocation
3. **Communicate** risk to the board or stakeholders
4. **Prioritize** remediation efforts

The detailed technical findings follow for the security team to act on.
